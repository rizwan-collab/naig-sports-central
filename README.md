# NAIG Sports Central 2027
### North American Ismaili Games — #CompeteConnectUnite

**Live:** https://rizwan-collab.github.io/naig-sports-central/

Complete tournament management system for NAIG 2027. Single HTML file, no install required.

## Access Levels

| Role | Password |
|------|----------|
| Spectator | Open — no login |
| Scorekeeper | PIN: `1234` (admin) · `2468` (basketball) · `1357` (soccer/volleyball) |
| Regional | `regional2027` |
| National Admin | `naig2027` |

## Features
- Live score board with real-time updates
- Pool standings, playoffs, medal table by region
- All 14 NAIG 2027 sports with official rules
- 58 teams · 40 athletes · 58 team games · 23 individual results · 20 medals
- 3-mode scheduling calculator
- Delete data by sport or region
- Scoresheet export and printable rulebook
- Mobile-friendly (bottom nav on phones)
- In-app help centre
