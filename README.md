### North American Ismaili Games — #CompeteConnectUnite

**Live URL:** https://rizwan-collab.github.io/naig-sports-central/

A complete tournament management system for NAIG 2027. Open in any browser — no install needed.

## What's Included

- Live score ticker + dedicated Live tab
- Pool standings, playoff brackets, medal table by region
- Individual sport results (badminton, chess, track, table tennis, tennis, dance, pickleball)
- 58 teams · 40 athletes · 58 team games · 23 individual sport results · 20 medals
- All 14 NAIG 2027 sports with official rules from naismailigames.com
- 3-mode scheduling calculator (fields needed / days needed / capacity)
- Delete data by sport or region
- Scoresheet export + rulebook (printable PDF)
- Mobile-friendly: bottom nav bar on phones, touch-optimized score entry
- AI help chat (bottom-right bubble)

## Tech
Single HTML file. React + Babel (CDN). Data stored in localStorage. JSON backup/restore.
